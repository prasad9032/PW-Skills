Text formatting tags

  1. Strong
    • it is used to identify text that is of greater importance than the surrounding text.
    • The strong tag supports Global Attributes and the Event Attributes.
  2. em
    • The em tag in HTML is a phrase tag and used to emphasize the text content.
    • em tag is used to stress emphasis the particular text within a sentence or phrase.
  3. Underline
    • Add an annotation that tells us something about the selected text itself, not about the content or message of the text.
    • The u element was originally used to identify text that should be underline
  4. del
    • it is traditionally used to track changes to a document and highlight removals from the document
    • The del tag defines text that has been deleted from a document.
