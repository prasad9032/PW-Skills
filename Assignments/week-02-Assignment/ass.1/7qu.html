Emmet

Emmet doesn’t have a predefined set of available tag names, you can write any word and transform it into a tag:

There many types of nesting operators in emmet it can be used according to the user need

Some of the nesting emmet
  1. ">"

    it is used to place the tag inside the main tag or the starting tag

  2. "+" sibling

    it is used to add the tag after the main tag it is just like sibling the words

  3. "*"multiplication

    it is used to give the multipel tag by multiplying with the given number

  4. () grouping

    Parenthesises are used by Emmets’ power users for grouping subtrees in complex abbreviations

  5. ID and classes

    = # it is used to add the id and =.classname will give the name of the class automatically
