list of semantic tags

  1. article: This tag is used to define a self-contained piece of content that can be distributed independently from the rest of the page.
  2. section: This tag is used to define a section of a document, such as a chapter or a header.
  3. header: This tag is used to define the header of a document or a section
  4. footer: This tag is used to define the footer of a document or a section.
